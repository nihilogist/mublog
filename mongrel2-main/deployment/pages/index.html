Hello World!

μblog is a micro-blog written to test the mongrel2 webserver, and a small scale implementation of a microservice architecture.

Why even do this?

Microservice envy. Or, in this case, because I think it is worth building something on a small scale before even thinking about using it on a large scale. And because the way mongrel works means that I can try adding in new components in different languages, just to see how they work, without needing to write a whole project in them.
